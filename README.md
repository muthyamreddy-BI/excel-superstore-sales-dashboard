#  Superstore Sales Dashboard (Excel)

##  Project Overview
This project is an **interactive Excel dashboard** built to analyze Superstore sales data across regions, categories, segments, and time.

The dashboard helps business users track **key KPIs**, identify high-performing regions and categories, and understand sales and profit trends for better decision-making.

##  Tools & Techniques Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers & Timeline Filters
- Excel Formulas
- Data Cleaning & Formatting

##  Key KPIs
- Total Sales
- Total Profit
- Total Quantity Sold
- Total Discount Given

##  Key Insights
- Technology is the top-performing category (36% of total sales)
- West region generates the highest sales
- Consumer segment contributes the majority of revenue
- Standard Class shipping accounts for the highest sales volume
- Sales and profit show consistent growth from 2014 to 2017

##  Files Included
- Super_Store_Sales_Dashboard.xlsx – Interactive Excel dashboard
- Screenshots/ – Dashboard visuals
- Super_store_sales_Data

##  How to Use
1. Download the Excel file
2. Open in Microsoft Excel (2016 or later recommended)
3. Use slicers (Region, Category, Segment, Ship Mode, Year) to explore insights dynamically
